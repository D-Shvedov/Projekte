# This is my first learning project

This is a simple Ping-Pong game built with Python using the pygame library.

## Project characteristics

- **Name:** Ping-Pong  
- **Type:** Game  
- **Frameworks:** None
- **Language:** Python
- **Library:** pygame
- **Module:** None
- **Database:** None
- **Additional technology:** None  

## Description

This game is made for two players.  
One player uses the **A** and **D** keys, the other player uses the **left** and **right arrow** keys.  
The maximum score is 10.

## Requirements

- Python 3.10+
