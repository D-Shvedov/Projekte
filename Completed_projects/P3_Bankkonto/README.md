# This is my third learning project

This is a terminal-based bank account application where you can use the PowerShell terminal to perform different operations (without a database).

## Project characteristics

- **Name:** Bankaccount  
- **Type:** Console Application  
- **Language:** Python  
- **Frameworks:** None
- **Library:** None  
- **Modules:** None 
- **Database:** RAM  
- **Additional technology:** None  

## Description

This application allows users to perform simple bank account operations in the terminal, such as checking the balance, depositing money, and withdrawing money.

## Requirements

- Python 3.10+
