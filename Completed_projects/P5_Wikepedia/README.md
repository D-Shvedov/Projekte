# This is my fiveth learning project

This is Wikepedia, which can show articles und make something as a magazine. People can add neue article themself

## Project characteristics

- **Name:** Wikepedia  
- **Type:** Local Web Application  
- **Language:** Python  
- **Frameworks:** None
- **Library:** tkinter  
- **Modules:** socket, threading
- **Database:** JSON-Datei  
- **Additional technology:** Server, Graphical User Interface  

## Description

This desktop application provides a graphical interface that allows multiple users to connect to each other over a network and exchange information in real time.  
It uses sockets for network communication and threading to handle multiple connections without freezing the user interface.

The goal of this project is to learn how to build networked applications with a GUI, manage parallel processes, and implement a basic client-server architecture in Python.
